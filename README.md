##### instalar las dependencias de laravel

Instalar las dependencias de laravel
```sh
$ composer install
```

Instalar las dependencias de vista 
```sh
$ cd public
$ npm install
```

ejecutar servidor artisan
```sh
$ php artisan serve
```
luego ir a la URL
127.0.0.1:8000/

y probar la app :)
